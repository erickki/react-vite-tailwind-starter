# React Vite Tailwind CSS Starter

## Descrição

Este repositório funciona como um starter kit e backup base para iniciar projetos React com JavaScript usando Vite. Ele já inclui toda a configuração inicial do TailwindCSS e o plugin Tailwind Animated, oferecendo uma estrutura pronta para ser clonada ou copiada diretamente para novos projetos.

O objetivo é facilitar a criação de apps modernos, mantendo sempre um template atualizado, otimizado e com todas as dependências compatíveis entre si. Isso elimina a repetição de configurações e acelera o início de qualquer desenvolvimento, garantindo padronização e velocidade.

## Funcionalidades

- React + JavaScript configurado via Vite.
- TailwindCSS totalmente instalado e funcional.
- Tailwind Animated integrado ao projeto.
- Estrutura inicial organizada para novos projetos.
- Configurações otimizadas para desenvolvimento rápido.
- Pronto para copiar e iniciar um novo repositório.
- Mantido atualizado para garantir compatibilidade entre versões.

## Versões

```
React: 19.2.0
Tailwind CSS: 4.1.17
Tailwind CSS - Animated: 2.0.0
Vite: 7.2.2
```

## Vantagens

- **Agilidade:** comece um projeto React em segundos.
- **Setup completo:** não precisa repetir instalações ou configurações.
- **Compatibilidade garantida:** sempre atualizado com versões que funcionam bem juntas.
- **Padrão de organização:** mantém consistência entre múltiplos projetos.
- **Tailwind Animated incluso:** facilita a criação de animações modernas.
- **Base personalizável:** pode ser adaptado a diferentes necessidades.
